# Screw Counter AI

Screw Counter AI is a sleek, modern web application that uses machine learning and computer vision to automatically detect and count screws within your images. It works seamlessly by interacting with a Roboflow serverless workflow and outputs a beautiful, structured analysis dashboard directly in your browser.

## Features

- **Drag and Drop Interface:** Quickly upload images by dragging them into the application or selecting them directly from your file system.
- **Client-Side Processing:** Images are previewed and securely converted to Base64 format locally in your browser before being sent to the AI model.
- **Premium UI:** Designed with modern aesthetics including dark mode, glassmorphism, responsive hover states, smooth CSS animations, and vibrant gradient typography.
- **Detailed Analytics Dashboard:** Upon completing the analysis, the app renders:
  - **Screws Detected:** An exact numerical count calculated directly from the AI's detected bounding boxes.
  - **AI Summary:** A natural language analysis directly generated by Google's Gemini models reviewing the object detection output.
  - **Visualization:** A rich, highlighted image illustrating exactly where the screws were detected using bounding boxes.

## How to Use

Because Screw Counter AI relies entirely on standard front-end technologies (Vanilla JS, HTML, and CSS), there is no complex build process required!

1. Clone or download this repository to your local machine.
2. Navigate to the project folder.
3. Double-click on `index.html` to open it in your preferred web browser.
4. Enter your Roboflow API key in the input field at the top and click **Save** (it will be securely persisted in your browser for future use).
5. Drag and drop an image containing screws onto the upload section, and click **Count Screws**.
6. Wait for the API to process the image and enjoy your results dashboard!

## Technologies Used

- **HTML5** (Structure)
- **Vanilla CSS3** (Responsive, premium styling without heavy frameworks)
- **Vanilla JavaScript (ES6+)** (Async/Await APIs, DOM manipulation, FileReader)
- **Roboflow Inference API** (Handling the computer vision workflow and object detection modeling)

## Security & API Keys

This application uses a secure "Bring Your Own Key" (BYOK) approach to handle API credentials purely on the front-end. Your Roboflow API key is **never** hardcoded into the source code, securing the application for pushing to public Git repositories (like GitHub) or deploying as a static site. 

Instead, the application dynamically prompts you for the API key via the web interface. Once provided, it is securely cached into your browser's `localStorage`, avoiding the need to continuously re-enter it, while keeping it strictly on your personal device.
